Commonly Found Issues


1. My client is having graphical issues! How do I fix this?

This issue occurs commonly if the client isn't using OpenGL, to do this, you have to go into Settings in Studio. To access Settings you have to go into Tools and you should find the settings button, once your in settings then go into Rendering and scroll down until you find 'GraphicsMode' and change the it to the following below which is meant for the client that your using:

2008: OpenGL stable

2009-2010: OpenGL legacy

2011-2012: Direct3D

2. Why is the CoreGui in 2011-2012 not working when I create or open an existing map! How do I fix this?

Due to RClient loading the CoreGui scripts locally, you have open studio, create a new map or load an existing one and then go into Tools and then ExecuteScript. Now go to the content folder of the client and you should see a file called 'guifix.lua' just double click on it and it will fix the Gui.

3. My character isn't spawning in 2009/2010! How do I fix this?

Go to your appdata folder, you can do this by pressing Windows key and R at the same time.

Type %appdata% then press enter

Delete all the files in the Roblox folder in Appdata Local (Don't delete versions if you want to keep Modern Studio and Client)

Delete rbxcsettings in Local Low
